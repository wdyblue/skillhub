# SkillHub Roadmap

## v0.1.0：最小可运行版本

- Tauri + React + SQLite 基础工程。
- 中文侧边栏和基础页面。
- 扫描目录管理。
- `SKILL.md` 扫描入库。
- 全部技能卡片列表。
- Skill 详情页。
- 基础质量评分。
- 基础归档。

## v0.2.0：整理能力增强

- 分类管理页：新增、编辑、删除、颜色设置。
- 标签系统：新增标签、批量打标签、按标签筛选。
- Skill 备注字段。
- 批量归档。
- 批量修改分类。

## v0.3.0：重复检测完整化

- 完全重复组：hash 相同。
- 名称相似组：v1、v2、backup、final、copy 等规则。
- 内容相似组：Jaccard / cosine / token overlap。
- 推荐保留项。
- 建议归档项。
- 忽略重复组。

## v0.4.0：技能体检报告

- 说明过短检测。
- 结构不完整检测。
- 长期未更新检测。
- 疑似旧版本检测。
- 推荐优先整理清单。
- 体检报告导出。

## v0.5.0：GitHub 能力预留实现

- 绑定远程 skill 仓库。
- 从 GitHub 导入 skill。
- 检查 skill 更新。
- 本地与远程 diff。
- 更新前自动备份。

## v1.0.0：稳定版

- 跨平台构建。
- 完整文档。
- 稳定数据迁移。
- 可发布 GitHub Release。
