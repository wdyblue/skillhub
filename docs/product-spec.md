# SkillHub 技能管理器产品说明

## 定位

SkillHub 是一个中文版、本地优先、卡片式 AI Skill 资产管理器，用于管理本机安装的大量 AI skills，并把本地统一仓库同步到不同 AI 工具目录。

## 目标用户

- 本地安装了大量 Codex / ChatGPT / Claude / Hermes / Cursor skills 的开发者或创作者。
- 需要整理自建 skill、下载 skill、多版本 skill 和重复 skill 的用户。
- 希望先本地安全管理，再逐步接入 GitHub 更新、Marketplace 和团队共享的用户。

## 核心价值

- 把散落在不同目录的 `SKILL.md` 统一扫描入库。
- 用卡片方式展示名称、描述、分类、评分、使用次数和重复风险。
- 通过质量评分帮助识别说明过短、结构不完整的 skill。
- 通过 hash 和名称规则识别疑似重复或旧版本。
- 用归档替代删除，降低误操作风险。
- 通过工具目录检测和链接策略同步，把统一 Skill 仓库启用到 Codex、CodeBuddy、Claude Code 等工具。
- 通过 OpenAI 兼容 API 翻译英文/中文 Skill 名称、描述和摘要。

## 当前版本范围

已实现：

- 本地目录添加、移除、启用、禁用。
- 扫描包含 `SKILL.md` 的文件夹。
- SQLite 本地数据库。
- 内置中文分类。
- 全部技能卡片页。
- Skill 详情页。
- `SKILL.md` 预览。
- 复制、打开文件夹、打开文件。
- 状态归档。
- 基础统计。
- 分类管理。
- 自建 Skill 新建与导入。
- 统一 Skill 主仓库。
- 多工具目录检测。
- Skill 到工具目录的链接启用/禁用，以及工具 x Skill 分配矩阵。
- 同步体检与自动修复。
- AI 翻译 API 配置、测试连接和技能卡片一键翻译。
- 本地账号状态。
- Marketplace JSON 源管理、刷新、安装、更新、卸载和安装态复核。
- 本地云同步包导出/导入。
- 一键同步全部工具链接。

暂未实现：

- 标签编辑。
- GitHub 远程更新。
- 远程账号认证。
- 真正服务端云同步。
- Marketplace 评分、评论、审核和付费能力。
- embedding 语义相似度。

## 非目标

当前版本不做：

- 云同步。
- 服务端 marketplace。
- 自动删除本地 skill。
- 自动移动本地 skill。
- 强依赖 AI API。AI 翻译是可选配置。
- 团队协作。
