# SkillHub 技能管理器产品说明

## 定位

SkillHub 是一个中文版、本地优先、卡片式 AI Skill 资产管理器，用于管理本机安装的大量 AI skills。

## 目标用户

- 本地安装了大量 Codex / ChatGPT / Claude / Hermes / Cursor skills 的开发者或创作者。
- 需要整理自建 skill、下载 skill、多版本 skill 和重复 skill 的用户。
- 希望先离线管理，再逐步接入 GitHub 更新能力的用户。

## 核心价值

- 把散落在不同目录的 `SKILL.md` 统一扫描入库。
- 用卡片方式展示名称、描述、分类、评分、使用次数和重复风险。
- 通过质量评分帮助识别说明过短、结构不完整的 skill。
- 通过 hash 和名称规则识别疑似重复或旧版本。
- 用归档替代删除，降低误操作风险。

## v0.1.0 范围

已实现：

- 本地目录添加、移除、启用、禁用。
- 扫描包含 `SKILL.md` 的文件夹。
- SQLite 本地数据库。
- 内置中文分类。
- 全部技能卡片页。
- Skill 详情页。
- `SKILL.md` 预览。
- 复制、打开文件夹、打开文件。
- 状态归档。
- 基础统计。

暂未实现：

- 完整重复检测分组页。
- 标签编辑。
- 分类 CRUD 页面。
- GitHub 远程更新。
- AI 自动摘要和 embedding 相似度。

## 非目标

第一版不做：

- 账号系统。
- 云同步。
- 在线 marketplace。
- 自动删除本地 skill。
- 自动移动本地 skill。
- 强依赖 AI API。
- 团队协作。
